1 # Matbotics
2 Matbotics is a Arduino library to control [Matrix Robotics System controller].
3 Matbotics uses [Arduino I2C master library] to support _repeat start_,
4 because Wire library (issued with the Arduino IDE) does not.
5 
6 ### Features
7 
8 - Get battery level, controller status, version, etc.
9 - Control all servos
10 - Set speed of all motors
11 - Set motor target
12 - Read motors encoders value
13 - Timeout function
14 
15 ### Installation (Unix like systems)
16 
17 Open a terminal and run :
18 ```bash
19 $ cd <UserDirectory>/Arduino/libraries
20 $ git clone https://github.com/blasterbug/Matbotics.git
21 ```
22 
23 Now after (re)starting the Arduino IDE you shloud see Matbotics in Arduino

26 ### Exemple
27 
28 This exemple shows how to control a servo plugged to the port one
29 
30 ```arduino
31 // import libraries
32 #include <I2C.h>
33 #include <Matbotics.h>
34 
35 // use the controller
36 MTController ctlr;
37 
38 void setup()
39 {
40  // enable servos
41  ctlr.enableServos();
42  // switchoff timeout
43  ctlr.timeout( 0 );
44  // set a custom speed for the servo one
45  ctlr.servoOneSpeed( 5 );
46 }
47 
48 // some variables
49 int angle = 0;
50 
51 void loop()
52 {
53  // increase angle and stay in boundaries
54  angle = (angle + 10)%250;
55  // set the angle to the first servo
56  ctlr.servoOneAngle( angle );
57  delay( 600 );
58 }
59 ```

70 ### Connect Lego Mindstorms devices to Arduino boards
71 
72 
73 ![NXT clipped off wire](http://blasterbug.github.io/Camduino/NXT_black_wire.png "NXT clipped off wire")
74 
75 NXT uses RJ12 wires made like this :
76 
77 - white : analog
78 - black : ground
79 - red : ground
80 - green : 4.3V
81 - yellow : I2C clock line (`SCL`)
82 - blue : I2C data line (`SDA`)
83 
84 Connect the red and the black wire to Arduino ground pin, the yellow to a `SCL` pin
85 and the blue one to a `SDA` pin. On Arduino boards this pins are clearly identified.
